# API Router Permissions

This document lists all the permissions used across different router endpoints in the API.

## Attendance
- `list:attedance` - List all attendance records

## Balance Account
- `list:balances` - List balance accounts
- `show:balances` - View balance account details 
- `create:balances` - Create new balance accounts
- `edit:balances` - Edit balance accounts
- `delete:balances` - Delete balance accounts
