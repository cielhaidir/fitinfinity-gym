## 🚀 Instalasi & Deployment

### Prerequisites
- Node.js 18+ atau 20+
- PostgreSQL 14+
- Docker & Docker Compose (optional)
- MQTT Broker (Mosquitto) - included in docker-compose
- MinIO Server - included in docker-compose

### Quick Start dengan Docker (Recommended)

```bash
# 1. Clone repository
git clone https://github.com/yourcompany/fitinfinity.git
cd fitinfinity

# 2. Copy environment variables
cp .env.example .env

# 3. Edit .env dan sesuaikan dengan konfigurasi Anda
nano .env

# 4. Start semua services dengan docker-compose
docker-compose up -d

# 5. Run database migration
docker-compose exec app npx prisma migrate deploy

# 6. Seed initial data
docker-compose exec app npm run db:seed

# 7. Akses aplikasi di http://localhost:3000
# Default login: admin@fitinfinity.com / admin123
```

### Manual Installation

```bash
# 1. Clone repository
git clone https://github.com/yourcompany/fitinfinity.git
cd fitinfinity

# 2. Install dependencies
npm install

# 3. Setup database
# Buat database PostgreSQL dan update connection string di .env
createdb fitinfinity

# 4. Copy dan konfigurasi .env
cp .env.example .env

# 5. Run migration
npx prisma migrate deploy

# 6. Generate Prisma Client
npx prisma generate

# 7. Seed database
npm run db:seed

# 8. Start development server
npm run dev

# 9. Akses di http://localhost:3000
```

### Environment Variables Penting

```env
# Database
DATABASE_URL="postgresql://user:password@localhost:5432/fitinfinity"

# NextAuth
NEXTAUTH_SECRET="your-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"

# MQTT
MQTT_BROKER_URL="mqtt://localhost:1883"
MQTT_USERNAME="fitinfinity"
MQTT_PASSWORD="secure-password"

# MinIO (S3 Storage)
MINIO_ENDPOINT="localhost"
MINIO_PORT="9000"
MINIO_ACCESS_KEY="minioadmin"
MINIO_SECRET_KEY="minioadmin"

# Payment Gateways
MIDTRANS_SERVER_KEY="your-midtrans-key"
DOKU_CLIENT_ID="your-doku-client-id"

# Email
SMTP_HOST="smtp.gmail.com"
SMTP_PORT="587"
SMTP_USER="your-email@gmail.com"
SMTP_PASS="your-app-password"
```

### Production Deployment

```bash
# Build untuk production
npm run build

# Start production server
npm run start

# Atau dengan PM2 untuk process management
pm2 start npm --name "fitinfinity" -- start
```

### IoT Device Setup

```bash
# 1. Download Arduino IDE atau PlatformIO
# 2. Install library dari folder /lib
# 3. Configure WiFi credentials di sketch
# 4. Upload firmware ke ESP32
# 5. Device akan otomatis connect ke MQTT broker
# 6. Daftarkan device di admin panel FitInfinity
```
